Classes

FieldDescriptionAttribute

Display custom field description in data editor.

NodeActionAttribute

Display button that could execute custom function in data editor.

NodeColorTagAttribute

Display custom color tag in data editor. Default is transparent. Only can attach on Color.

NodeDescriptionAttribute

Display custom description in data editor. Only can attach on String.

NodeElementAttribute

Define child nodes in data editor. Only can attach on IEnumerable<UnityEngine.Object> and UnityEngine.Object.

NodeGroupedElementAttribute

Define nodes in data editor. Nodes will be grouped in a pseudo node. Only can attach on IEnumerable<UnityEngine.Object> and UnityEngine.Object.

NodeIconAttribute

Define this node's icon in data editor. Only can attach on UnityEngine.Texture and UnityEngine.Sprite.

NodeTitleAttribute

Display custom title in data editor. Only can attach on String.

OnionDataEditorGameObjectAgent

Because game object can not attach custom node attribute, use OnionDataEditorGameObjectAgent to display children node in node editor.

OpenWithOnionDataEditor

Asset default open with data editor if set true.
